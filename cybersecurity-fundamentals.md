# Cybersecurity Fundamentals

## What is Cybersecurity?
Cybersecurity focuses on protecting systems, networks, and data from unauthorized access or attacks.

## Common Threats
- Phishing attacks
- Malware
- Unauthorized access
- Social engineering

## Key Protections
- Multi-Factor Authentication (MFA)
- Regular system updates and patching
- Access control management
- User awareness and training

## Human Factor
Many security incidents occur due to user behavior, not system failure.

## Real-World Insight
Strong cybersecurity is not about having tools—it’s about using them correctly and consistently.
